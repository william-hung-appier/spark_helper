# Tauri macOS App Support

## Context/Background

- Chrome extension uses Side Panel API which is not supported in Arc and some other Chromium-based browsers
- Team members using Arc cannot access the Spark Helper tool
- Need a standalone macOS app that works outside the browser

## Solution

Added Tauri-based macOS app that wraps the existing extension UI:

- **No code duplication** - Tauri loads the same HTML/JS/CSS as the extension
- **Lightweight** - 4.7 MB app bundle (vs 150+ MB for Electron)
- **No runtime dependencies** - Team members just double-click to run

## Project Structure

```txt
spark_helper/
├── src-tauri/
│   ├── Cargo.toml          # Rust dependencies
│   ├── build.rs            # Tauri build script
│   ├── tauri.conf.json     # App configuration
│   ├── src/
│   │   └── main.rs         # Entry point (minimal)
│   └── icons/
│       ├── icon.png
│       └── icon.icns
├── dist/                   # Symlinks for Tauri frontend
│   ├── index.html → ../sidepanel.html
│   └── src → ../src
└── package.json            # Added tauri scripts
```

## Build Commands

| Command | Description |
|---------|-------------|
| `make tauri-dev` | Development mode with hot reload |
| `make tauri-build` | Production build |

## Build Output

| File | Size | Location |
|------|------|----------|
| `Spark Helper.app` | ~5 MB | `src-tauri/target/release/bundle/macos/` |
| `Spark Helper_x.x.x_aarch64.dmg` | ~2 MB | `src-tauri/target/release/bundle/dmg/` |

## Prerequisites (for building)

- Rust toolchain via mise: `mise use rust@latest`
- Tauri CLI: `cargo install tauri-cli`
- Xcode Command Line Tools

## Distribution

Share the `.dmg` file via Slack/Drive. No dependencies needed to run.
